Ini adalah proyek website profil pribadi yang saya buat sebagai bagian dari tugas akhir praktikum Pemrograman Web. Website ini berfungsi untuk memperkenalkan diri saya secara digital, menampilkan latar belakang pendidikan, minat, serta hobi saya dengan tampilan visual yang menarik dan mudah dinavigasi.

Rancang Bangun dan Tujuan

Website ini dirancang menggunakan HTML dan CSS dengan fokus pada tampilan yang bersih, modern, serta responsif di berbagai perangkat. Tujuan utamanya adalah sebagai representasi diri secara profesional dan personal yang menggambarkan kepribadian, minat, serta kemampuan dasar saya dalam membangun halaman web statis.

Struktur dan Fitur Halaman

Website ini terdiri dari beberapa bagian utama:

Header
Berisi logo dan navigasi sticky yang memudahkan pengunjung berpindah antarhalaman seperti Beranda, Tentang Saya, Keahlian, Kontak, dan Hobi & Minat.

Section Tentang Saya
Menjelaskan profil singkat, biodata diri (termasuk NPM: 2315061005),
Section Keahlian
Disusun dalam bentuk kartu dengan gaya CSS yang menarik dan efek hover modern, menampilkan kemampuan saya di bidang:

HTML5

CSS3

Section Baru: Hobi dan Minat
Bagian tambahan yang berisi informasi tentang aktivitas yang saya sukai di luar bidang akademik, seperti:

Fotografi dan Desain untuk melatih kreativitas visual.

Pemrograman Web sebagai eksplorasi teknologi baru.

Musik dan Traveling untuk menjaga keseimbangan antara produktivitas dan relaksasi.

Tampilan section ini didesain menggunakan layout fleksibel dengan ikon dan warna lembut agar lebih menarik secara visual.

Section Kontak
Berisi informasi kontak seperti alamat email, nomor telepon, dan tautan ke media sosial agar pengunjung dapat berinteraksi dengan mudah.

Footer
Ditampilkan di bagian bawah halaman dengan warna kontras lembut dan teks singkat sebagai penutup.

Perubahan Desain dan Peningkatan Visual

Dalam versi terbaru website ini, beberapa perbaikan telah dilakukan agar tampilannya lebih menarik:

Penambahan animasi halus (hover & transition) pada tombol dan kartu keahlian.

Penggunaan warna yang lebih modern dan konsisten pada seluruh halaman.

Penambahan section baru “Hobi dan Minat” untuk memperkaya konten pribadi.

Penyusunan ulang layout menggunakan grid dan flexbox agar lebih rapi dan responsif.

Peningkatan tampilan header dan navigasi agar terlihat profesional.

Kesimpulan

Website ini bukan hanya sebagai tugas akhir praktikum, tetapi juga sebagai latihan nyata dalam mengimplementasikan konsep dasar HTML dan CSS untuk membangun website pribadi yang fungsional dan estetis. Dengan adanya halaman tambahan dan pembaruan gaya, website ini menjadi lebih informatif dan mencerminkan karakter saya sebagai mahasiswa Teknik Informatika yang kreatif dan bersemangat untuk terus belajar.
